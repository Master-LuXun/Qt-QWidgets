3.Qtui_FramelessWidget 自定义标题栏(无边框窗体)
  特点:
  自动识别双击标题栏响应。
  无边框拉伸在windows下不抖动。
  在windows下具有移动到边缘半屏、移动到顶部全屏特性。
  解决mac系统上无边框最小化最大化失效的BUG。
  解决系统休眠后再次启动程序懵逼的BUG。
  解决有时候窗体重新显示的时候假死不刷新的BUG。
